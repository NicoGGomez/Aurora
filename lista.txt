Esto también puede agarrar números que no son volumen.

✔ Mejor:

re.search(r"(?:volumen\s*)?(\d{1,3})\s*%?", texto)

🎯 3. Mejor manejo de volumen (mucho mejor)
def manejar_volumen(texto):
    if "máximo" in texto or "maximo" in texto:
        set_volumen(100)
        return

    if "mutear" in texto:
        set_volumen(0)
        return

    porcentaje = obtener_porcentaje(texto)
    if porcentaje is not None:
        print(f"Acomodando volumen a {porcentaje}%")
        set_volumen(porcentaje)
🧠 4. Mejor regex de porcentaje
def obtener_porcentaje(texto):
    match = re.search(r"(\d{1,3})\s*%?", texto)
    if match:
        val = int(match.group(1))
        return max(0, min(val, 100))  # clamp
    return None
🎤 5. Optimizar reconocimiento de voz (MUY importante)

En vez de crear micrófono cada vez:

mic = sr.Microphone()

def escuchar():
    with mic as source:
        r.adjust_for_ambient_noise(source, duration=0.2)
        audio = r.listen(source)

    try:
        return r.recognize_google(audio, language="es-AR").lower()
    except sr.UnknownValueError:
        return ""
